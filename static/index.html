站点设置

                                
配置网站主页介绍、管理员入口地址、公开分享独立域名，以及 Cloudflare Turnstile 验证。

                            

                            

                                
                                
                            

                        


                        

                            

                                
主页与后台

                                

                                    
                                    
                                    
显示在根路径 / 的网站主页上。

                                

                                

                                    
                                    
                                    
访客打开主页时会先看到这段介绍，用来说明网站用途，不暴露后台入口。

                                

                                

                                    
                                    
                                    
必须是站内路径，例如 /admin 或 /safe-panel。修改后后台会立即切换到新地址。

                                

                                

                                    
                                    

                                        
                                        
                                        
                                    

                                

                            


                            

                                
公开分享域名

                                

                                    
                                    
启用后，公开分享链接必须通过指定域名访问；主站域名上的公开页会被关闭。

                                

                                

                                    
                                    
                                    
只填写域名，支持端口，例如 share.example.com 或 127.0.0.1:9000。

                                

                                

                                    
                                    
默认关闭。如果公开分享域名不是管理员后台同一个域名，建议开启并配置独立密钥；同域名时也可以继续共用管理员默认配置。

                                    
当前默认关闭。配置独立分享域名后，系统会根据域名是否一致给出建议。

                                

                                

                                    
                                    
                                

                                

                                    
                                    
                                    
开启后，公开分享页面会优先使用这里的独立密钥，与管理员默认 Turnstile 配置完全分离。

                                

                                

                                    
效果说明

                                    

                                        1. 邮件公开设置里的链接会自动替换成这个域名。

                                        2. 指定分享域名上只允许公开分享页面和它依赖的静态资源访问。

                                        3. 这样可以把后台入口和公开页面拆开，减少主站暴露面。
                                    

                                

                            


                            

                                
Cloudflare Turnstile

                                

                                    
                                    
                                    
管理员登录页和公开页面密码验证共用这一组 Site Key 与 Secret Key。

                                

                                

                                    
                                    
                                    
如果留空保存，Turnstile 会被视为未配置，两个页面开关都会自动关闭。

                                

                                

                                    
                                    

                                        
                                        
                                    

                                    

                                        X
                                        受到其他独立密钥影响已被禁用
                                    

                                    
公开页面只有在该邮箱已设置访问密码时，才会出现 Turnstile 验证框。

                                

                                

                                    为了减少页面跳动，登录页和公开密码页都预留了固定高度的验证区域。请确保你的 Turnstile 小组件已允许管理员域名和分享域名访问。
